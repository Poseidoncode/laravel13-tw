## 驗證方法

1. **檔案清單：** 使用 `find` 工具列出 `src/pages/docs/` 下所有 `.astro` 檔案
2. **翻譯標記檢查：** 使用 `grep` 工具搜尋「此頁面為」「本文翻譯自」「繁體中文」「翻譯自」等關鍵字
3. **中文字元驗證：** 使用 `grep` 搜尋 Unicode CJK 統一漢字範圍（U+4E00–U+9FFF），確認每個檔案包含實際中文內容
4. **差異比對：** 使用 `comm` 工具比對完整檔案清單與含標記的檔案清單，確認無遺漏
